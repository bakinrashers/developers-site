Open Source

It’s not just our journalism that’s open. Many of our projects are hosted on GitHub and we actively contribute to the open source community.

Active projects

Here are a few of our favourite open source projects. For a list of all publicly available repos, see the Guardian’s GitHub account.

  • Frontend

    Source for the Guardian’s next-generation responsive website.

  • Scribe

    A rich text editor framework for the web platform, with patches for browser inconsistencies and sensible defaults.

  • Riff Raff

    The Guardian’s Scala-based deployment library designed to help automate deploys.

  • sass-mq

    A Sass mix-in that helps manipulating media queries in an elegant way.

  • guss

    A collection of CSS and Sass bower components re-usable across multiple Guardian web products.
